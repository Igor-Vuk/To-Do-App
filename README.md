# Test

Start project:

1.run mongod (runs on localhost:27017)
2.`npm install`
3.`npm start`
4.open localhost 9000

Api routes:

- GET `/api/todos`
- POST `/api/todos`
- GET `/api/todos/:id`
- PUT `/api/todos/:id`
- DELETE `/api/todos/:id`
